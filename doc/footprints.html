Footprints endpoint

This section gives you information about the footprints used by projects in Goteo.org.

Footprints are a sort of meta-categorization for projects in Goteo which include them in one "big" area of influence, Ecological footprint, Social footprint and Democratic footprint.

Each category, social commitment and SDG (Sustainable Development Goals) belongs to one or more footprints.

/footprints/

Obtains a list of footprints.

curl -i --basic --user "user:key" https://api.goteo.org/v1/footprints/

Filters available:

The standard set of filters applies to this endpoint with these particulars:

Attribute Type Description
node string Filter by individual node(s). Multiple nodes can be specified. Restricts the list to the footprints used in projects assigned in that nodes
call string Filter by individual matchfunding call(s). Multiple calls can be specified. Restricts the list to the projects originally created in that call(s)
project string Filter by individual project(s). Multiple projects can be specified. Restricts the list to the footprints used in that projects
from_date string Filter from date. Ex. "2013-01-01". Restricts the list to the footprints used in projects created between that dates
to_date string Filter until date.. Ex. "2014-01-01". Restricts the list to the footprints used in projects created between that dates
category integer Filter by category. Multiple categories can be specified. Restricts the list to the social commitments matching with that category(ies)
sdg integer Filter by Sustainable Development Goal. Multiple SDGs can be specified. Restricts the list to the social commitments matching with that SDG(s)
social_commitment integer Filter by social commitment. Multiple social commitments can be specified. Restricts the list to the social commitments specified
lang string Get results by specified lang. Multiple langs can be specified
location number Filter by project location (Latitude,longitude,Radius in Km). Restricts the list to the footprints used in projects geolocated in that area

Response values:

ResponseFootPrint
Attribute Type Description
items array ... ☞ FootPrint
meta ... ☞ MetaFootPrint
FootPrint
Attribute Type Description
description string FootPrint short description
icon-url string URL for the image resource associated with this social commitment
id string FootPrint unique identifier
name string FootPrint name
total-projects integer Number of published projects in the FootPrint
total-users integer Number of users interested in the FootPrint
MetaFootPrint
Attribute Type Description
from_date string Only if filtering by from_date
lang array Only if filtering by lang
location array Only if filtering by location
node array Only if filtering by node
project array Only if filtering by project
to_date string Only if filtering by to_date
total integer Total elements available for the collection
