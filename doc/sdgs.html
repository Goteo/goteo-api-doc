Sustainable Development Goals endpoint

This section gives you information about the Sustainable Development Goals (SDGs) used by projects in Goteo.org.

SDGs are another way to categorize projects in Goteo based The 2030 Agenda for Sustainable Development, adopted by all United Nations Member States in 2015.

Each SDG belongs to a one or more footprints and social commitments.

/sdgs/

Obtains a list of sdgs.

curl -i --basic --user "user:key" https://api.goteo.org/v1/sdgs/

Filters available:

The standard set of filters applies to this endpoint with these particulars:

Attribute Type Description
node string Filter by individual node(s). Multiple nodes can be specified. Restricts the list to the sdgs used in projects assigned in that nodes
project string Filter by individual project(s). Multiple projects can be specified. Restricts the list to the sdgs used in that projects
from_date string Filter from date. Ex. "2013-01-01". Restricts the list to the sdgs used in projects created between that dates
to_date string Filter until date.. Ex. "2014-01-01". Restricts the list to the sdgs used in projects created between that dates
category integer Filter by category. Multiple categories can be specified. Restricts the list to the SDGs matching with that category(ies)
social_commitment integer Filter by social commitment. Multiple social commitments can be specified. Restricts the list to the social commitments specified
footprint integer Filter by project footprint. Multiple footprint can be specified. Restricts the list to the sdgs matching that footprint(s)
lang string Get results by specified lang. Multiple langs can be specified
location number Filter by project location (Latitude,longitude,Radius in Km). Restricts the list to the sdgs used in projects geolocated in that area

Response values:

ResponseSDG
Attribute Type Description
items array ... ☞ SDG
meta ... ☞ MetaSDG
SDG
Attribute Type Description
description string SDG short description
icon-url string URL for the image resource associated with this social commitment
id string SDG unique identifier
name string SDG name
total-projects integer Number of published projects in the SDG
total-users integer Number of users interested in the SDG
MetaSDG
Attribute Type Description
from_date string Only if filtering by from_date
lang array Only if filtering by lang
location array Only if filtering by location
node array Only if filtering by node
project array Only if filtering by project
to_date string Only if filtering by to_date
total integer Total elements available for the collection
