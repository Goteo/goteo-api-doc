Social Commitments endpoint

This section gives you information about the social commitment used by projects in Goteo.org.

Social Commitments an alternative way of categorization for projects in Goteo.

Each social commitment can belong to a one or more SDG (Sustainable Development Goals) and footprints.

/socialcommitments/

Obtains a list of socialcommitments.

curl -i --basic --user "user:key" https://api.goteo.org/v1/socialcommitments/

Filters available:

The standard set of filters applies to this endpoint with these particulars:

Attribute Type Description
node string Filter by individual node(s). Multiple nodes can be specified. Restricts the list to the socialcommitments used in projects assigned in that nodes
call string Filter by individual matchfunding call(s). Multiple calls can be specified. Restricts the list to the projects originally created in that call(s)
project string Filter by individual project(s). Multiple projects can be specified. Restricts the list to the socialcommitments used in that projects
from_date string Filter from date. Ex. "2013-01-01". Restricts the list to the socialcommitments used in projects created between that dates
to_date string Filter until date.. Ex. "2014-01-01". Restricts the list to the socialcommitments used in projects created between that dates
category integer Filter by category. Multiple categories can be specified. Restricts the list to the social commitments matching with that category(ies)
social_commitment integer Filter by social commitment. Multiple social commitments can be specified. Restricts the list to the social commitments specified
sdg integer Filter by project Sustainable Development Goal. Multiple SDGs can be specified. Restricts the list to the social commitments matching that SDG(s)
footprint integer Filter by project footprint. Multiple footprints can be specified. Restricts the list to social commitments with that footprint(s)
lang string Get results by specified lang. Multiple langs can be specified
location number Filter by project location (Latitude,longitude,Radius in Km). Restricts the list to the socialcommitments used in projects geolocated in that area

Response values:

ResponseSocialCommitment
Attribute Type Description
items array ... ☞ SocialCommitment
meta ... ☞ MetaSocialCommitment
SocialCommitment
Attribute Type Description
description string SocialCommitment short description
icon-url string URL for the image resource associated with this social commitment
id string SocialCommitment unique identifier
name string SocialCommitment name
total-projects integer Number of published projects in the SocialCommitment
total-users integer Number of users interested in the SocialCommitment
MetaSocialCommitment
Attribute Type Description
from_date string Only if filtering by from_date
lang array Only if filtering by lang
location array Only if filtering by location
node array Only if filtering by node
project array Only if filtering by project
to_date string Only if filtering by to_date
total integer Total elements available for the collection
